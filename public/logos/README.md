# Logos de clientes/parceiros (placeholders)

Esta pasta guarda os logotipos exibidos na seção **Provas rápidas** da Home.

## Como atualizar

1. Substitua os arquivos `.svg` pelos logotipos aprovados.
2. Preserve os nomes e proporções originais para evitar *layout shift*.
3. Os SVGs devem ter `width` e `height` definidos e não conter `scripts`.
4. Otimize o código removendo metadados e atributos desnecessários.
5. Verifique o contraste para garantir acessibilidade em fundo claro.

> Enquanto os logotipos reais não são aprovados, mantemos placeholders simples.
